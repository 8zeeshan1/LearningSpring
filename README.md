# Spring Boot Learning Journey 

Welcome to this repository!

This repository documents my journey of learning **Spring Boot** through hands-on practice. Rather than just reading documentation or watching tutorials, I'm building small examples, experimenting with different concepts, and gradually expanding my understanding of the framework.

## What you'll find here

As I continue learning, I'll keep adding examples and implementations covering topics such as:

* Spring Boot fundamentals
* Dependency Injection (IoC)
* Beans and Component Scanning
* REST APIs
* Spring Data JPA
* Hibernate
* Database Integration
* Validation
* Exception Handling
* DTOs
* Security
* Testing
* And many more Spring Boot concepts

This repository will be updated regularly as I explore new topics and improve existing implementations.
